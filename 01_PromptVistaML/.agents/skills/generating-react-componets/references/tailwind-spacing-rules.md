# Tailwind Spacing Rules

## Vertical Rhythm
- Use consistent vertical margins and padding to create a rhythm.
- Large sections: `py-16`, `py-20`, or `py-24`.
- Section gaps: `space-y-16` or `space-y-20` on the parent container.

## Component Spacing
- Internal padding for cards/panels: `p-5`, `p-6`, or `p-8`.
- Margins below headings: `mb-4` or `mb-6`.
- Spacing between adjacent inline elements: `space-x-2` or `gap-2` in flex containers.

## Typography Scaling
- Headings: `text-2xl` to `text-5xl` (or larger for heroes `text-7xl`).
- Body text: `text-sm` or `text-base`.
- Monospace metadata/eyebrow text: `text-[10px]` or `text-xs` with `tracking-widest uppercase`.
